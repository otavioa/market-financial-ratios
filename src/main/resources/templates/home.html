Market Financial Ratios


Endpoints

All available endpoints. Enjoy!

Endpoint Description Test
/market-ratio/all Retrieve all available tickers and their ratios. Test
/market-ratio?<params> Retrieve tickers by setting 'types', 'tickers' and 'ratios' parameters. Test
/market-ratio/etfs/{ticker} Retrieve ticker ratios from ETF (only works for Brazilian ones), you need to specify one by one. Test

Available Types

Types that you can use as a parameter for 'types' on /market-ratio?types=X endpoint.

Key Type Description
1 ACOES Retrieve tickers by filtering Brazilian stocks.
1 FIIS Retrieve tickers by filtering Brazilian reits.
1 STOCKS Retrieve tickers by filtering American stocks.
1 REITS Retrieve tickers by filtering American reits.

Available Ratios

Those are all available ratios that you can use for parameter 'ratios' on /market-ratio?<params> endpoint.

Key Ratio Description
1 PL P/L (P/E for stocks) - Price/Earnings.
2 ROE Return on Equity
3 LPA LPA (EPS for stocks) â€“ Earnings Per Share.
4 PVP P/VP (P/BV for stocks)- Price to Book Value.
5 VPA VPA (BVPS for stocks) - Book value per share.
6 DY % Dividend Yield (MAT).

Tips

  • The application will return '-' for unavailable values.
  • The application might return 400* or 500* in case of error.
  • It's possible to match values with its origin (StatusInvest), all data come from there. Just access Status Invest.
  • For endpoint /market-ratio?<params> you'll need to specify at least one parameter of 'types' or 'tickers'.
